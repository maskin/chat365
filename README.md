# 音声放送制御アプリ

## 1. 概要

社内・校内放送をスケジュールに従って自動実行するための音声制御アプリケーションです。
音声入力によって放送内容をタスクとして登録し、指定日時に自動で放送することで、担当者の負担軽減と伝達の正確性向上を目指します。

## 2. MVPの主な機能

- **音声による放送登録:** 話した内容をテキスト化し、放送タスクとして登録します。
- **スケジュール実行:** 指定された日時に、登録内容を音声合成して放送します。
- **緊急放送:** 即時に放送が必要な場合のための、割り込み放送機能です。

## 3. 技術スタック（予定）

- **フロントエンド:** HTML, CSS, JavaScript
- **バックエンド:** Node.js (Express) / Python (FastAPI)
- **データベース:** SQLite / PostgreSQL
- **クラウドサービス:** Google Cloud Speech-to-Text / Text-to-Speech API
